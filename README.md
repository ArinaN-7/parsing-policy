# Парсер политических партий

Программа для извлечения списка политических партий и ссылок на их документы с сайта Министерства юстиции РФ.

## Описание

Программа парсит HTML-страницу "Политические партии" с сайта minjust.gov.ru и извлекает:
- Наименование политической партии
- Ссылку на документ партии (PDF/страница)

## Подготовка HTML файла

1. Откройте страницу: https://minjust.gov.ru/ru/pages/politicheskie-partii/
2. Сохраните страницу как `minjust.html` (Файл → Сохранить как → Веб-страница, полностью)
3. Положите файл `minjust.html` в папку с программой

## Результат работы

Программа создает файл parties.json со списком партий в формате json:
[
  {
  "name": "Всероссийская политическая партия «ЕДИНАЯ РОССИЯ»",
    "doc_url": "https://minjust.gov.ru/ru/documents/7767/"
  },
  ...
]

## Структура проекта

parse_parties.py - основной скрипт парсера
minjust.html - сохраненная HTML-страница
parties.json - результат парсинга (создается автоматически)
requirements.txt - зависимости
README.md - документация
.gitignore - исключает виртуальное окружение и временные файлы
